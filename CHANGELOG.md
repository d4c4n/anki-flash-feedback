## [1.0.0] - 2026-05-30

### Added

- Initial release.
- Per-answer-button (Again / Hard / Good / Easy) color and on/off toggle.
- Configurable hold duration, fade-out duration, and peak opacity.
- GUI config dialog (Tools → Add-ons → Config).
- Config migration from legacy `fail_color`/`pass_color` and intermediate `ease_colors` schemas.
- Non-blocking, click-through overlay; handles rapid card reviews safely.
